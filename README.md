# node-adp-workforce-now
Node.js library to interact with ADP's Workforce Now API.

## installation

```bash
npm install 
```

## npm

### Scoping

```bash
$ npm init --scope=@craibuc
```

### publish

- increment the `version` property in the `package.json` file
- run `npm publish --access public`
